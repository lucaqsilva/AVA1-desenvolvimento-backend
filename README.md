# AVA1 - Desenvolvimento Backend

Projeto desenvolvido para a disciplina de Desenvolvimento Full Stack, com foco na parte de backend.

O objetivo deste projeto é implementar funcionalidades de servidor, rotas, APIs e integração entre frontend e backend.

## Tecnologias utilizadas

- Node.js
- Express
- JavaScript

## Objetivo acadêmico

Este repositório possui finalidade acadêmica para estudos e desenvolvimento prático da disciplina.
